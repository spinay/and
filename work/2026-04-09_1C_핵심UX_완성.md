## 상세 내용

### 1C-1: 청약 기록 추가/수정 폼 화면

**신규 파일**: `lib/presentation/subscription_form/subscription_form_screen.dart`

- 종목 선택 (BottomSheet) → 증권사/수량/증거금 입력
- 자동 계산: 증거금 (`수량 × 공모가 × 비율`), 환불금 (`증거금 - 배정 × 공모가`), 수익 미리보기
- 진행 상태 ChoiceChip 선택 (applied → allocated → refunded → listed → sold)
- 섹션 점진 펼침: 기본정보 → 배정단계 → 매도단계
- `initialFocus` 파라미터로 home CTA에서 직접 해당 섹션으로 진입
- 수정 모드에서 삭제 기능 (확인 다이얼로그)
- Router: `/subscription/new?ipoId=...`, `/subscription/edit/:id?focus=allocation|sale`
- my_ipo FAB → 신규 추가, 카드 탭 → 편집
- home CTA: 청약시작→신규, 환불→배정편집, 상장→매도편집

### 1C-2: 날짜 기반 상태 자동 전이

`SubscriptionRepository.autoTransition()` 메서드 추가.

- 앱 시작 시 청약 기록을 스캔, catalog의 환불일/상장일과 비교
- applied/allocated → refunded (환불일 이후)
- refunded → listed (상장일 이후)
- main.dart에서 시드 후 즉시 호출

### 1C-3: 로컬 알림

**신규 파일**: `lib/core/services/notification_service.dart`

- `flutter_local_notifications` + `timezone` (Asia/Seoul) 초기화
- 관심종목 기반 3종 알림 스케줄:
  - 청약 시작 D-1 오전 9시 — "내일 청약 시작"
  - 환불일 당일 오전 9시 — "오늘 환불금 확인"
  - 상장일 당일 오전 9시 — "오늘 상장일"
- 과거 날짜 자동 스킵
- 앱 시작 시 watchlist 기반 재등록 (cancelAll → 재스케줄)

### 1C-4: 수익 화면 고도화

- 요약 카드: 총 투입금, 승률(%), 건당 평균 수익 추가
- "진행중" 섹션 추가 (매도 전 청약 현황 + 투입금 합계)
- 빈 상태 안내 개선 ("내 청약 탭에서 기록을 추가해보세요")
- 매도 기록 있으면 월별 차트 + 종목별 리스트 표시
